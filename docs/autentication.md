# Авторизация и аутентификация

1. На главной странице XXX нажмите **Получить токен**.
2. Укажите актуальный адрес электронной почты.
3. Проверьте электронную почту на наличие [письма](*ключ_термина) с токеном.
4. Перейдите по адресу <https://api.documentat.io/editor/>.
5. Нажмите **Authorize**.
6. В поле _Value_ введите токен.
7. Нажмите **Authorize**.

[*ключ_термина]: Если письмо не пришло, проверьте папку **СПАМ**.
