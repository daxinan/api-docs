# Быстрый старт

1. Откройте терминал компьютера и нажмите сочетание клавиш [WIN+R](*ключ_термина) .
2. Введите запрос:
```curl
curl -X GET "https://api.documentat.io/api/dev/users?limit=10" -H "accept: application/json"
```
где: 
`curl` - терминальная утилита для веб-запросов;
`-X` - флаг, указывающий метод запроса (в данном случае GET);
`"https://..."` - адрес веб-ресурса;`
`-H` - передаваемый заголовок в формате `"ключ: значение"`.
3. Получите ответ:
```curl
{"error":401,"message":"Неавторизованный доступ"}
```
Данный ответ означает, что запрос выполнен успешно. Но, чтобы выполнить операцию, необходимо [авторизоваться](autentication.md).

[*ключ_термина]: Команда применима в Windows 10.
